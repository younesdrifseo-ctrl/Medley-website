# Mobile First Designer — MÉDLËY

## Breakpoints
- 320px: minimum supported width
- 768px: tablet
- 1024px: desktop
- 1920px: max-width containers

## Mobile Rules
- Touch swipe on gallery (native scroll)
- Tap feedback: active:scale(0.97)
- No animations > 300ms on mobile
- cursor:none only on @media(hover:hover)
- Safe-area-inset for iPhone notch
- Call button visible bottom-left on mobile
